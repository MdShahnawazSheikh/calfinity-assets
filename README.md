## 🌐 CDN Usage

All assets are served via jsDelivr CDN for fast and reliable delivery.

### Base URL:

```
https://cdn.jsdelivr.net/gh/MdShahnawazSheikh/calfinity-assets/
```

### Example:

```
https://cdn.jsdelivr.net/gh/MdShahnawazSheikh/calfinity-assets/badges/v1/perfect-week.png
```

## ⚙️ Usage in App

These assets are intended to be used in the Calfinity Mobile app via network image loading (e.g., CachedNetworkImage).

---

## 🧠 Notes

* All images are optimized for mobile performance
* File names follow kebab-case naming convention
* Do not rename or restructure files without updating app references
* Versioning is handled via folder structure (e.g., `v1`, `v2`)
